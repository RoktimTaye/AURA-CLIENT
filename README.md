# Aura Client

[![Built with TanStack Start](https://img.shields.io/badge/Built%20with-TanStack%20Start-FF4154?style=flat-square&logo=react)](https://tanstack.com/start)
[![React 19](https://img.shields.io/badge/React-19-61DAFB?style=flat-square&logo=react)](https://react.dev/)
[![Tailwind CSS v4](https://img.shields.io/badge/Tailwind%20CSS-v4-06B6D4?style=flat-square&logo=tailwindcss)](https://tailwindcss.com/)

**Aura** is a next-generation price transparency platform designed to empower communities with real-time, verified data on local commodity prices. Built with a focus on speed, accuracy, and user experience, Aura leverages Machine Learning (ML) to detect price anomalies and forecast future trends.

---

## Key Features

- **📍 Real-time Price Tracking:** View and report live prices for groceries, fuel, and other essentials based on your location.
- **🛡️ ML-Powered Verification:** Automated anomaly detection (Z-Score) identifies and flags suspicious price reports to prevent spam.
- **📈 Price Forecasting:** Advanced time-series analysis using **Meta's Prophet** to predict price trends for the next 7 days.
- **🗳️ Community Voting:** A democratic verification system where users upvote or downvote entries to ensure data integrity.
- **📊 Admin Dashboard:** Comprehensive analytics and moderation tools for managing data streams and user activity.
- **⚡ Server-Side Rendering (SSR):** Powered by TanStack Start for lightning-fast initial loads and superior SEO.

---

## 🛠 Tech Stack

### Frontend (Client & SSR)
- **Framework:** [TanStack Start](https://tanstack.com/start) (React 19)
- **Routing:** [TanStack Router](https://tanstack.com/router) (Type-safe, file-based)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/), [shadcn/ui](https://ui.shadcn.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/), [GSAP](https://gsap.com/)
- **State Management:** [Zustand](https://zustand-demo.pmnd.rs/)
- **Data Fetching:** [TanStack Query](https://tanstack.com/query)
- **Charts:** [Recharts](https://recharts.org/)
- **Forms:** [React Hook Form](https://react-hook-form.com/) + [Zod](https://zod.dev/)

### Backend & ML (Integration)
- **API Framework:** [FastAPI](https://fastapi.tiangolo.com/) (Python)
- **Database:** [PostgreSQL](https://www.postgresql.org/) (Neon Serverless)
- **ML Engine:** Meta Prophet (Forecasting), NumPy (Anomaly Detection)
- **Deployment:** [Cloudflare Pages/Workers](https://workers.cloudflare.com/)

## ⚙️ Getting Started

### Prerequisites
- **Node.js**: v20.x or higher
- **npm**: v10.x or higher

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo/aura-client.git
   cd aura-client
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the development server with Hot Module Replacement (HMR):
```bash
npm run dev
```
The application will be accessible at `http://localhost:8080`.

### Building for Production
The project is optimized for Cloudflare Pages/Workers:
```bash
npm run build
```
This generates the `dist/` directory containing both the client assets and the server-side worker.

## ⚙️ Architecture Highlights

### TanStack Start & SSR
Aura uses **TanStack Start** to bridge the gap between client and server. It utilizes `createServerFn` for secure server-side operations and Route Loaders to fetch data before the page renders, eliminating "loading spinners" for critical data.

### ML Pipeline
The backend integrates a Python-based ML engine:
1. **Anomaly Detection:** Every price submission is checked against historical standard deviations.
2. **Forecasting:** When viewing an item, the system fetches predictions generated by a Prophet model, providing users with actionable insights.

For more technical details, refer to:
- [Backend & ML Architecture](./BACKEND_AND_ML_ARCHITECTURE.md)

## ☁️ Deployment

This application is designed to run on the **Cloudflare Edge Network**.
- **Worker Configuration:** Managed via `wrangler.jsonc`.
- **SSR Handling:** Cloudflare Workers handle the SSR logic, providing global low-latency responses.

To preview the production build locally using Wrangler:
```bash
npm run preview
```
